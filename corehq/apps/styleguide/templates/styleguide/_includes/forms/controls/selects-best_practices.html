Select2 vs. Standard Select

You always should use a Select2 widget over a standard select widget when the options require the user to scroll extensively through the select list to find the right option. Conversely, it's overkill to use a Select2 widget when there are only a few options.

You should use a Select2 Async widget when the list of data is so long that it is more performant to filter the list first before sending the data to the user.
