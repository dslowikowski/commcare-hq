Standard Multi Select

Many options may be chosen instead of one.

Select2

Use the Select2 approach if you want to allow searching as the user types text (like a typeahead), but want to limit the final choice to returned results (or have any control over the final formatting of the user's choice). The Select2 API is well documented.

Select2 Async

This differs from the regular Select2 since the data is not taken from the option generated inside the select tag generated on page load. Instead, data is loaded asynchronously via angular-js (documentation forthcoming) and .select2() is applied to an input type="hidden" tag, rather than a select tag.
